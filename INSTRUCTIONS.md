# Instructions

1. Install:
```bash
npm i -S three-vreffect-module
```

2. Use:
```javascript
import VREffect from 'three-vreffect-module';
const effect = new VREffect(renderer);
effect.setSize(window.innerWidth, window.innerHeight);
```
